# *VILLAGE-BOUNDARY-EXTRACTOR*
## *Motivation:*
Sometimes we need only few village shapefiles, but it takes time or some times system crashes due to heavy data to extract such village boundaries from the whole state data, so This code helps to extract the village Boundary by Village Attributes and Filter Options and converts to the Shape file. This code is useful mostly for the low-configuration systems as GIS Softwares so much time for loading or sometimes its crashes.

## *Data Sources:*
you can download few of the Village Boundary Geojson files from [*Data {Meet}*](https://projects.datameet.org/indian_village_boundaries/) Website
Here are the available Data from the above website currently
* Bihar
* Karnataka
* Kerala
* Goa
* Gujarat
* Maharastra
* Sikkim
* Odisha
